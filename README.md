borg
====

Sane Machine Management

To run without upgrade abilities
`sudo ./daemon.sh [port] [user-to-execute-commands-as]`

To with with upgrade ability
`sudo ./daemon.sh [port] [user-to-execute-commands-as] bootstrap`
